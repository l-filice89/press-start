Where does the Catalog live?

The shelf has been the whole app — “one screen, the user never navigates away.” Epic 7 adds a second destination for the ~490-game PS+ catalog, so that rule breaks either way. The only question is how you get there. Same content in all three; only the chrome differs.

Recommended no new chrome zone

Two tabs in the header you already have. Both destinations always visible, one tap, nothing hidden. The FAB stays what it is — chores only.

zero new chrome
